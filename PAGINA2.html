A vantagem de fazer sites para mobiles, é que a grande maioria dos dispositivos realmente bons para acessar a web, tem um bom supote a Padrões Web. Ter conforto para navegar nestes aparelhos é importantíssimo e por isso os fabricantes de sistemas e browsers tentam sempre mantê-los atualizados


Temos os seguintes browsers e motores de renderização:



Webkit Devices com Android instalado ou iPhones e iPod Touch (com Mobile Safari).


Opera Mobile e Opera Mini O Opera é um dos browsers mais "perfeitos" que existem para mobiles. Ele tem um bom suporte aos Padrões.


Blackberry O browser utilizado nos Blackberrys são muito bons. Bem melhores que o Internet Explorer.


Browsers S60 da Nokia São baseados em Webkit também, como o browser do iPhone e do Android, mas geralmente a versão do webkit é antiga, o que atrapalha um pouco se você quiser fazer algumas coisas mais elaboradas.


Internet Explorer Mobile Ele não tem bom suporte a CSS, é bastante lento e não tem uma séries "features" que os outros browsers tem, como o Opera. Contudo, com o lançamento do Windows Phone 7 isso tende a mudar um pouco. O Windows Phone 7 utiliza um browser com características e suporte aos padrões referentes a alguma versão do Internet Explorer 7 e 8.



O WebKit é considerado como a melhor engine para browsers mobiles. É rápido e suporta grande parte das especificações de HTML, CSS e Javascript. Mesmo assim, é preciso testar os sites nos outros sistemas e para isso temos os emuladores, embora nem todos tem versões para vários sistemas operacionais. O simulador de iPhone só tem para Mac. O de Android tem para Mac e para Windows. O de Blackberry e os da Nokia só tem para Windows.




Android:

Funciona em Macs e Windows. Mas são complicados de usar. Nada que alguns minutos lendo tutoriais e artigos pela internet não resolvam. A grande vantagem é que você consegue customizar o aparelho onde você vai testar o website.


iPhone/iPod Touch:

O mais fiel de todos os emuladores de mobiles que já vi. Tudo o que você vê na tela, é exatamente o que verá no aparelho. O problema é que só funciona no Mac. Os emuladores alternativos para windows ou outros sistemas não são fiéis.


Opera Mobile e Opera Mini:

Fizeram uma atualização geral estes últimos tempos. A vantagem é que ele funciona on-line. Nada de fazer download.


Blackberry:

Só funciona em Windows. A vantagem é que dá para escolher versões anteriores. Mesmo assim, sugiro que sempre nivele pela última versão.
